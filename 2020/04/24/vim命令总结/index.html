# VIM 命令总结

# 进入编辑模式的一些复合命令

复合命令 等效长命令 等效无副作命令 描述
C c$ D 删除从当前光标到行尾的内容
s cl x 删除光标下的一个字符
S ^C dd 删除光标所在行
I ^i 将光标移动到行首，并进入插入模式
A $a 将光标移动到行尾，并进入插入模式
o A<CR> 在光标所在行的下方插入一行，并进入插入模式
O ko 在光标所在行的上方插入一行，并进入插入模式

这些命令与 . 命令结合使用有奇效。

# 可重复的操作及如何回退

目的 操作 重复 回退
作出一个修改 {edit} . u
在行内查找下一个指定字符 f{char}/t{char} ; ,
在行内查找上一个指定字符 F{char}/T{char} ; ,
在文档中查找下一处匹配项 /pattern<CR> n N
在文档中查找上一处匹配项 ?pattern<CR> n N
执行替换 😒/target/replacement & u
执行一系列修改 qx{changes}q @x u
