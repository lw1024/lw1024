# Nginx 编译安装

# 编译安装的一般步骤

  1. tar -xvf [name].tar.gz
  2. cd [name]
  3. ./configure --prefix=[path]
  4. make
  5. make install

# 编译安装 nginx 所需的工具和库

  1. gcc
  2. pcre
  3. zlib
  4. openssl

# 开始安装

./configure --prefix=/app/nginx --sbin-path=/usr/bin/nginx --with-http_stub_status_module --with-http_ssl_module --with-pcre=../pcre-8.43 --add-module=../ngx_http_proxy_connect_module

参数说明：

  • --prefix 用于指定 nginx 编译后的安装目录
  • --add-module 为添加的第三方模块，此次添加了 fdfs 的 nginx 模块
  • --with..._module 表示启用的 nginx 模块，如此处启用了 http_ssl_module 模块

# 启动、停止、重启

  1. 启动 nginx

    nginx

    可通过 ps -ef | grep nginx 查看 nginx 是否已启动成功

  2. 停止 nginx

    nginx -s stop

  3. 重新启动

    nginx -s reload
