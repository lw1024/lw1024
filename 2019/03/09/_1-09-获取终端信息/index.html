# 获取终端信息

  1. 获取终端的行数和列数

    tput cols
tput lines

  2. 打印出当前终端名

    tput longname

  3. 将光标移动到坐标（100，100）处

    tput cup 100 100

  4. 设置终端背景色

    tput setb n  # n 可以在 0～7 之间取值

  5. 设置文本前景色

    tput setf n  # n 可以在 0～7 之间取值

  6. 设置文本样式为粗体

    tput bold

  7. 设置下划线的起止

    tput smul
tput rmul

  8. 删除从当前光标位置到行尾的所有内容

    tput ed

  9. 不显示密码输入内容的例子

    #!/bin/sh

tput bold
echo -e "Enter password: "
stty -echo #禁止将输出发送到终端
read password
stty echo  #允许发送输出
echo
echo Password read.

    选项 -echo 禁止将输出发送到终端，而选项 echo 则允许发送输出。
